O que é o GengarDex?

O GengarDex é uma aplicação web que permite buscar informações completas sobre Pokémon, incluindo tipos, descrições, evoluções e análises estratégicas de batalha. Além disso, o projeto apresenta sugestões de times ideais para diferentes jogos da franquia.

Como funciona?

Buscar Pokémon: Digite o nome de um Pokémon na barra de pesquisa para visualizar suas informações.

Visualizar dados: São exibidos o tipo, descrição, cadeia de evolução, forças, fraquezas, resistências e imunidades.

Favoritar: É possível salvar Pokémon favoritos para acesso rápido posteriormente.

Explorar jogos: A seção de jogos apresenta times recomendados para diferentes versões, com informações detalhadas de cada Pokémon.

Tecnologias utilizadas:

HTML: Estrutura da aplicação.

CSS: Estilização da interface e organização dos elementos.

JavaScript: Lógica da aplicação, manipulação de DOM e consumo de API.

PokéAPI: Fonte de dados dos Pokémon.

Como usar este projeto:

Clone o repositório: Faça o download ou clone para sua máquina.

Abra o arquivo index.html: Execute diretamente no navegador.

Utilize a busca: Digite o nome de um Pokémon para começar.

Explore os jogos: Clique no botão de jogos para visualizar os times sugeridos.

Observações:

Dados dos jogos: Os times estão definidos no arquivo dadosJogos.js e podem ser facilmente editados.

Imagens: As sprites dos Pokémon são carregadas dinamicamente via URLs externas.

Armazenamento: Os favoritos são salvos localmente no navegador utilizando localStorage.
